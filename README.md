# Daily Tracker CLI (Go)

CLI-инструмент для отслеживания ежедневного состояния:
- настроение
- энергия
- концентрация

Проект позволяет быстро записывать данные и получать статистику прямо из терминала.

---

## ✨ Features

- Добавление записи через CLI
- Поддержка флагов (`--mood`, `--energy`, `--focus`)
- Интерактивный ввод при отсутствии флагов
- Сохранение данных в CSV
- Статистика (средние значения)
- Фильтрация последних записей (`--last`)
- Валидация данных (0–5)
