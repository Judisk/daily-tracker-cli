# Daily Tracker CLI (Go)

Простая CLI-программа для отслеживания состояния:

* настроение
* энергия
* концентрация
* приём таблеток

## Features

* Добавление записи через CLI
* Сохранение данных в CSV
* Просмотр статистики
* Простая в использовании

## Example

```bash
go run main.go
```

Пример ввода:

```
Введите настроение (0-5): 3
Введите энергию (0-5): 2
Введите концентрацию (0-5): 4
```

## Stats mode

```bash
go run main.go stats
```

## Tech

* Go
* CSV (стандартная библиотека)

Проект сделан как инструмент для отслеживания состояния и анализа самочувствия по дням.
